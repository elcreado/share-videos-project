## Video Share Project

This is a video sharing platform built with Next.js, Supabase, and Tailwind CSS.
It's was made just for learning purposes.

## Tech Stack

- Next.js
- Supabase
- Tailwind CSS

## Getting Started

If you want to run this project for one reason, you can do following the next steps:

1. Clone this repository
2. Install dependencies
3. Set up environment variables
4. Run the development server

Easy. right?
